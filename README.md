Apple Financial Analysis Dashboard

An interactive and dynamic Power BI dashboard built to analyze Apple’s financial performance using key business metrics.

 Tools Used
 SQL           — data cleaning and preparation
 Power BI      — dashboard design and data visualization
 Power Query   — data transformation
 DAX           — calculated measures and KPIs
 Data Modeling — table relationships and analysis

 Key KPIs
- Revenue
- ASP (Average Selling Price)
- Units Sold

 Dashboard Features
- Interactive slicers for dynamic filtering
- KPI cards to track Revenue, ASP, and Units Sold
- Trend analysis across time periods
- Clear charts and visualizations for business insights
- Data model created to support accurate reporting

 Project Workflow
1. Cleaned and prepared the data using SQL.
2. Imported the cleaned data into Power BI.
3. Transformed data using Power Query.
4. Created relationships through data modeling.
5. Developed DAX measures for Revenue, ASP, and Units Sold.
6. Designed an interactive dashboard with visuals, KPI cards, and slicers.
